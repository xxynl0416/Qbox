云同步（GitHub Gist）

把所有书的进度、笔记、重点同步到 GitHub Gist。换个电脑/浏览器粘贴 token 就能拉回来。

怎么获取 Token？
  1. 打开 GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 点 "Generate new token (classic)"
  3. 勾选 gist 权限（其它不用勾）
  4. 设置过期时间（建议 90 天）
  5. 点 Generate token，复制 ghp_... 开头的字符串
  6. 粘到上面输入框，点 "保存配置"

⚠️ Token 只存在本地浏览器，不会上传到任何服务器。
⚠️ 同步的 Gist 默认是私有的（只有你能访问）。
💡 想换无 GitHub 的方案？可以把 token 字段填 Gitee Personal Access Token，把 API URL 改成 https://gitee.com/api/v5/gists（自行修改代码）。
